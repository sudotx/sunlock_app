# Boilerplate NextJS + Typescript + Firebase
You can use this a starter project for your web-app.

## Features
It includes the following tech-stack:
- Typescript 5
- NextJs 14
- Firebase 9 (authentication)
- Headuless UI
- Tailwind CSS

## Hosting
It's preferred choice for hosting is Vercel.
